Create Map Generator

Map Generator is a module used to generate Business Maps automatically.
 The XML rows are generated after the user inserts the parameters he is going to use for a specific functionality.

In Map Generator you can :

Create and save a new map
Edit an existing map
Each map has a history, so you can navigate to all modification that are made.
Each Business Map has its own design.
The design ( GUI ) I have implemented is user friendly.
There are more than 20 map types which can be generated by this module.
There's no need to write XML rows , now you can use **Map Generator ** to generate the Business Maps
that we use in coreBOS

(

    `name` varchar(250) NOT NULL,
    `text` varchar(250) NOT NULL
) ENGINE=InnoDB DEFAULT CHARSET=latin1;

INSERT INTO `mapgeneration_httpresponsetype` ( `name`, `text`)

,('atom', 'atom') 
,('rss', 'rss') 
,('json', 'json')
